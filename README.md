# 🏢 偷懶辦公室 — 虛擬辦公室 Dashboard

## 概念
一個前端網頁，以辦公室平面圖呈現團隊運作狀態：
- 每個房間 = 一個部門
- 房間內有員工座位
- 需求流動用連接線呈現（類似 n8n）
- 即時顯示誰在處理什麼任務、進度到哪

## 角色層級
```
凱凱（老闆）
  ↓ 下達命令
Kevin 小幫手（秘書）
  ↓ 分派任務
各部門組長（Clawdbot Agents）
  ↓ 分配執行
部門員工（Sub-agents / 工具）
```

## 技術選型（v1 簡潔版）
- **前端框架**: React + Vite（或純 HTML/CSS/JS 做 MVP）
- **平面圖渲染**: HTML Canvas 或 SVG
- **連線動畫**: SVG path + CSS animation（類 n8n 流動線）
- **資料來源**: Clawdbot Gateway API（sessions/agents 狀態）
- **部署**: 本機開發，localhost 預覽

## 功能規劃

### Phase 1（MVP）
- [ ] 靜態辦公室平面圖
- [ ] 各房間/部門顯示
- [ ] 員工頭像 + 名稱 + 狀態燈號
- [ ] 任務卡片：誰在做什麼
- [ ] 需求流動連線動畫

### Phase 2
- [ ] 即時連接 Clawdbot API 拉取 session 狀態
- [ ] 任務歷史紀錄
- [ ] 點擊員工看詳細任務
- [ ] 老闆下指令 → 秘書分派的動畫

- [ ] 語音指令
- [ ] 行動裝置適配
